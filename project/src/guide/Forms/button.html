CRUD 사용 위치 (그리드, 테이블 아래)
- 기본은 페이지 최하단에 사용
- 좌측, 우측, 좌우측에 사용 가능
- 버튼의 Class는 "btn"를 사용합니다.
- addClass로 색깔 및 모양을 사용합니다. (blue, purple, bLine, gLine, 등)
- 정렬기능 있는 태그를 빼지 않습니다. (alignL, alignC, alignR)
모바일 Class 규칙
반응형에서의 정렬은 align class 태그에서 class를 변형 및 추가하며, 부모나 자식태그를 가급적 사용하지 않습니다.
*샘플 1: (첫번째 버튼라인 : 정렬만 변경)
*샘플 2: (두번째 버튼라인 : 버튼1개당 100%로 변형)
*샘플 3: (세번째 버튼라인 : 모든 버튼이 1라인에 꽉차게 변형)
- class btnBox 태그라인에 여러가지 변형 class 형태를 부여해 더 다양하게 만들 수 있습니다.
